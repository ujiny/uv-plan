## 동기
사용자에게 자외선 지수 정보를 알려주기 위함.

## 예상 타겟층
- 주로 야외 활동 하는 사람
- 자외선 차단제 사용자
    - 사용 여부 고민 하는 사람
    - SPF 30/50 중에 고민 하는 사람
- 피부 민감한 사람(피부질환자)
