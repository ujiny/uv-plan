# 기능 (Feature) 리스트
## 홈 화면
- 자외선 지수 정도에 따라 배경화면으로 표현 
  - 안전 / 낮음 / 보통 / 나쁨 / 매우 나쁨
- 위치 정보
- 현재 자외선에 따른 대처 방안 제시
- 공유 기능
  - 아이폰 기본 공유 기능 
- 예보 기능
  - 내일 / 모래
- 위치 설정 기능
  - 위치 설정 화면으로 이동 
- 설정 버튼 

## 위치 설정 화면
- 검색 기능
- 최근 검색한 위치 기록
  - 리스트 상단에 표시

## 설정 버튼 기능 (왼쪽 상단)
- 자외선 정보 출처
  - 기상청(3~11월), 한국환경공단, OpenWeatherMap(12,1,2월)
- 세계보건기구(WHO)기준 자외선 지수
  - (0~2낮음 3~5 보통 6~7 높음 8~10 매우높음 11이상 매우위험)
- 불편 사항, 개선점 제시 등 개발자에게 메일 기능
- 위젯기능 설정
- 알림 푸시 설정
  - 시간 지정
- 애플 워치 연동 방법
